## Deployment Instructions

### Deploy to Vercel (Recommended)

1. **Install Vercel CLI** (if you haven't already):
   ```bash
   npm install -g vercel
   ```

2. **Navigate to the project folder**:
   ```bash
   cd zikr-game-deploy
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Deploy to Vercel**:
   ```bash
   vercel
   ```

5. **Follow the prompts**:
   - Login to Vercel (create account if needed)
   - Set up and deploy your project
   - Get your shareable link!

### Alternative: Deploy via Vercel Dashboard

1. Go to [vercel.com](https://vercel.com)
2. Sign up/Login with GitHub, GitLab, or Bitbucket
3. Click "Add New Project"

## Game Mechanics

- **Focus Mode**: Tap moving zikr phrases to earn points
- **Speed Progression**: Game speed increases every 15 seconds
- **Life System**: Game ends after 5 consecutive misses
- **Dynamic Unlocking**: New phrases unlock at point milestones
- **Golden Highlights**: Newly unlocked phrases appear in gold for first 3 times
- **Word Distribution**: 90% short (2-word), 5% medium (3-word), 2% longer phrases
